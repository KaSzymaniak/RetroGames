# RetroGames

Projekt `RetroGames` to prosta kolekcja gier retro dostępna w przeglądarce. W repozytorium znajdziesz frontend oraz backend, a w szczególności jedną z klasycznych gier:

- `Saper` – popularna gra logiczna, w której odkrywasz pola i oznaczasz miny.

W przyszłości można tu dodać kolejne gry, aby rozszerzyć bibliotekę retro tytułów.
